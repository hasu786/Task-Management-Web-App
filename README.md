# TaskFlow - Task Management Web App

A fully functional task management web application built with HTML, CSS, and JavaScript. Perfect for showcasing in your portfolio!

## Features

- **Task Management**: Create, read, update, and delete tasks
- **Project Organization**: Group tasks by projects (Personal, Work, Shopping, etc.)
- **Priority Levels**: Set task priority (Low, Medium, High)
- **Due Dates**: Set deadlines for tasks with visual indicators
- **Task Filtering**: Filter by project, status, and due date
- **Search Functionality**: Quickly find tasks
- **Progress Tracking**: Visual progress bar and statistics
- **Local Storage**: All data saved in browser's local storage
- **Import/Export**: Backup and restore your tasks
- **Responsive Design**: Works on desktop and mobile devices

## File Structure

task-manager/

## How to Use

1. **Add a Task**: Click the "Add Task" button
2. **Edit a Task**: Click the edit icon on any task
3. **Complete a Task**: Check the checkbox next to a task
4. **Delete a Task**: Click the trash icon on any task
5. **Filter Tasks**: Use the sidebar to filter by project or status
6. **Search Tasks**: Use the search box to find specific tasks
7. **Add Projects**: Create custom project categories
8. **Export/Import**: Backup your tasks or import from another device

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Flexbox, Grid, responsive design, CSS variables
- **JavaScript (ES6)**: Local Storage, DOM manipulation, event handling
- **Font Awesome**: Icons
- **Google Fonts**: Poppins font family

## Installation

No installation required! Simply:

1. Download all files to a folder
2. Open `index.html` in your web browser
3. Start managing your tasks!

## Browser Compatibility

Works in all modern browsers:
- Chrome (recommended)
- Firefox
- Safari
- Edge

## Customization

This app is designed to be easily customizable:

1. **Colors**: Modify the CSS variables in `style.css`
2. **Projects**: Edit the default projects in `script.js`
3. **Features**: Add or remove features by modifying the JavaScript
